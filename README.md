# Ray Wong — Personal Portfolio

A premium, responsive single-page portfolio built with React, Tailwind CSS, and Lucide React icons.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- **React 19** + **Vite 6**
- **Tailwind CSS 4**
- **Lucide React** icons
- **Space Grotesk** + **Inter** typography

## Sections

1. Hero — headline, subheadline, dual CTAs, key stats
2. Unicorn Value Proposition — hybrid enterprise + AI edge
3. Featured Portfolio — sports tech & digital product cards
4. Corporate Track Record — timeline with enterprise achievements
5. Skills & Ecosystem Matrix — scannable capability tags
6. Contact & Footer — form, LinkedIn, email, WeChat

## Customization

- Edit content in `src/App.jsx`
- Adjust colors in `src/index.css` (`@theme` block)
- Update contact links (LinkedIn, email) in the Contact section
